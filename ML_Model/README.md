# STGNN Traffic Prediction

## Project Structure

```
stgnn_project/
├── training/
│   └── train_stgnn.py          # Full training pipeline
├── deployment/
│   ├── app.py                  # FastAPI inference server
│   ├── model.py                # STGNN model architecture
│   ├── recommendation.py       # Severity scoring & strategy logic
│   ├── requirements.txt
│   ├── Dockerfile
│   │
│   │   # ---- generated by train_stgnn.py ----
│   ├── stgnn_traffic_model.pth
│   ├── x_scaler.pkl
│   ├── event_scaler.pkl
│   ├── y_scaler.pkl
│   ├── A_road.npy
│   ├── A_traffic.npy
│   └── A_event.npy
└── datasets/
    ├── link1.xlsx              # ← YOU provide this
    ├── traffic_with_dissipation.csv
    ├── linkwise_predictions.csv
    └── traffic_management_results.csv
```

## Step 1 — Train

```bash
# Place link1.xlsx inside datasets/
cd training
python train_stgnn.py
```

This saves the model weights + all 3 scalers + adjacency matrices into `deployment/`.

## Step 2 — Run API locally

```bash
cd deployment
pip install -r requirements.txt
python app.py
# API available at http://localhost:8080
```

## Step 3 — Docker / Cloud Run

```bash
cd deployment
docker build -t stgnn-api .
docker run -p 8080:8080 stgnn-api
```

## API Endpoints

| Method | Path       | Description              |
|--------|------------|--------------------------|
| GET    | /health    | Health check             |
| POST   | /predict   | Run prediction + get severity + strategy |

### /predict request body
```json
{
  "X": [[[...]]],   
  "E": [[[...]]]    
}
```
Shape: `X` → `(SEQ_LEN=4, N_nodes=8, 11)`, `E` → `(4, 8, 6)`
